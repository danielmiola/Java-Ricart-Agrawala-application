Essa implementação do RA utiliza 3 programas Java (P1, P2, P3) para simular
um grupo de processos.

Essa implementação utiliza um package, FilaProcessos, com classes auxiliares.

Para testar essa implementaçao, abra 3 terminais navague, em cada um deles, para a pasta onde se encontram os arquivos P1, P2, P3.
Após a compilação, execute um programa em cada terminal e observe os resultados. Mesmo com um sleep de 10 segundos em cada programa, é aconselhavel 
executa-los rapidamente para evitar erros de socket. Em cada um dos processos (programas), serão mostradas as mensagens ao serem liberadas para a aplicação, bem como o tempo de envio da mensagem, para confirmar que a implementação rode certo.

Os arquivos dos processos contêm trechos de código comentados que, ao serem retirados dos comentários, mostram quando mensagens são enviadas ou recebidas.
